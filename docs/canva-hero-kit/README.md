# Canva Hero Banner Kit

This kit is for rebuilding and editing the homepage hero banners inside Canva without being locked into one flattened image.

## Recommended Canva Custom Size

- Main website hero: `2400 x 900 px`
- Lighter working file: `1600 x 600 px`
- Keep the final export ratio at `8:3`

Use `2400 x 900 px` if the banner is for the website. That matches the current homepage hero ratio.

## Safe Layout Zones

- Left copy zone: `x 90 to 1180`
- Right artwork zone: `x 1360 to 2320`
- Top safe padding: `90 px`
- Bottom safe padding: `90 px`

Avoid putting important text too close to the far left and right edges because carousel arrows can sit over the banner on the site.

## Font Direction

- Heading: `Fraunces`
- Body / chips / buttons: `Plus Jakarta Sans`

If those are not available in Canva, use:

- Heading fallback: `Cormorant Garamond` or `DM Serif Display`
- Body fallback: `Manrope` or `Montserrat`

## Folder Structure

- `backgrounds/`: blank banner canvases
- `layers/`: photo and decorative pieces you can move independently
- `references/`: current full banner examples for visual guidance
- `canva-kit-preview.png`: quick sheet showing the kit at a glance

## Important Note About Transparency

Some PNG layers may show black around the artwork in simple image preview tools. That black area is transparency, not part of the artwork. In Canva it will drop out correctly.

This matters most for:

- `layers/easter-top-cluster.png`
- `layers/easter-lower-cluster.png`
- `layers/easter-eggs-daisies-cutout.png`

## Suggested Canva Build Order

1. Create a new Canva design at `2400 x 900 px`.
2. Upload the matching file from `backgrounds/`.
3. Add the matching files from `layers/`.
4. Add your text, chips, and CTA buttons.
5. Compare with the matching file in `references/` if you need layout direction.
6. Export as `PNG`.

## Quick Copy Direction

If you want the banners to stay consistent:

- Use 1 strong headline
- Use 1 short supporting sentence
- Use 2 to 3 small chips max
- Use 1 CTA button

That keeps the website hero clean and avoids the overcrowded look you were trying to remove.
